One Thousand and One Deaths

This repository contains the working manuscript for An Amateur's Guide to Last Things, Ever After, a novel-in-fragments composed of death stories, endings, disappearances, transformations, and returns.

The project moves across forms and genres. It may include prose, poetry, lists, notes, dialogue, figures, cartoons, multilingual passages, and other experiments in style. Its subject is not only the death of people, but also the death of thoughts, traditions, aesthetics, languages, habits, stars, objects, and states of being. Some endings are literal. Some are comic. Some are quiet. Some turn out to be beginnings in disguise.

The manuscript is being written in Markdown and intended for web-friendly rendering through HTML generation.
